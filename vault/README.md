# AI 오피스2 - 에이전트 팀 Vault

이 Obsidian vault는 AI 에이전트 팀의 **장기 기억** 및 **작업 위키** 역할을 합니다.

## 팀 소개

| 에이전트 | 이름 | 역할 |
|---------|------|------|
| PM | 지우 | 프로젝트 조율 및 사용자 소통 |
| 리서처 | 하늘 | 정보 수집 및 분석 |
| 개발자 | 준 | 코드 작성 및 설계 |
| 작가 | 소라 | 문서 및 콘텐츠 작성 |

## 폴더 구조

- **프로젝트/** - 프로젝트별 위키 (README, 진행상황, 산출물)
- **작업/** - 개별 Task 노트 (진행중, 완료)
- **지식베이스/** - 리서처가 축적한 지식 (기술, 트렌드, 참고자료)
- **코드베이스/** - 개발자 산출물 (스니펫, 아키텍처)
- **콘텐츠/** - 작가 산출물 (블로그, 문서, 기획)
- **에이전트-로그/** - 에이전트별 실행 기록
- **템플릿/** - 노트 생성용 템플릿

## 사용 방법

1. `npm start`로 에이전트 팀 CLI 실행
2. PM 지우에게 원하는 작업을 자연어로 입력
3. 에이전트들이 협력하여 작업 수행 후 이 vault에 결과 저장
4. Obsidian 앱에서 이 폴더를 열어 결과물 확인
